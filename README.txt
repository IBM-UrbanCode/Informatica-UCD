Overview
Informatica specializes in data management and gives their users powerful methods to 
access this data through Informatica Power Center’s Client Tools, Repositories, and Servers.

The UCD Informatica Plugin is an automation plugin that connects to specific 
repositories and directly access and merge data across remote systems within your organization.

The Informatica plug-in includes the following steps:

    Create Static Deployment Group
    Create Dynamic Deployment Group
    Deploy Deployment Group
    Roll Back Deployment Group
    ValidateDeployment Group
    Apply Label
    Run PMREP Command
    Import Objects

Compatibility
	The IBM UrbanCode Deploy automation plug-in works with Informatica Version 9.6 and later.
	This plug-in requires version 6.1.1 or later of IBM UrbanCode Deploy.

Installation
	No special steps are required for installation. See Installing plug-ins in UrbanCode Deploy.
